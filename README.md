# 🌤 Weather App

Приложение для просмотра погоды в любом городе.

## 🚀 Функционал
- Поиск города
- Отображение температуры
- Иконки погоды
- Геолокация пользователя
- Обработка ошибок

## 🛠️ Технологии
- HTML
- CSS
- JavaScript
- REST API (OpenWeather)

## 📦 Установка
1. Вставить API ключ в script.js
2. Открыть index.html

## ⚠️ Важно
Требуется API ключ с openweathermap.org

## 💡 О проекте
Используются асинхронные запросы (fetch, async/await) и работа с API.
